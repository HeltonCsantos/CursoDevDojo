# ☕ Maratona Java

Este repositório contém meus estudos e exercícios realizados durante o curso **Maratona Java**, focado no aprendizado da linguagem Java e dos principais conceitos de programação orientada a objetos.

O objetivo deste repositório é consolidar o conhecimento adquirido nas aulas e registrar a evolução no aprendizado da linguagem.

---

## 📚 Conteúdos Estudados

Durante o curso foram abordados diversos conceitos fundamentais da linguagem Java, entre eles:

- Sintaxe básica da linguagem
- Tipos primitivos
- Estruturas condicionais
- Estruturas de repetição
- Arrays
- Orientação a Objetos (POO)
  - Classes e objetos
  - Encapsulamento
  - Herança
  - Polimorfismo
  - Sobrescrita de métodos
- Modificadores de acesso
- Construtores
- Blocos de inicialização
- Associação entre classes
- Enumerações (Enum)
- Tratamento de exceções
- Boas práticas de organização de código
